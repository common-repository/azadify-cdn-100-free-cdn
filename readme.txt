== Description ==

Typically, you have to pay a lot of money every month to use a CDN. No longer. <a href="http://azadify.com/cdn" target="_blank">Azadify CDN</a> is a 100% free, fast, and full-featured CDN that offers you unlimited bandwidth and HTTPS (SSL) support... for free. The Azadify CDN Wordpress plugin helps you to quickly and easily integrate Azadify's 100% free CDN with Wordpress websites.

Learn more about Azadify CDN at http://azadify.com/cdn.

**What does it do**

Azadify CDN Wordpress plugin automatically and easily adds Azadify's 100% free CDN to your Wordpress website.

**Azadify CDN Features**

* 100% free CDN
* Unlimited bandwidth
* Full HTTPS (SSL) support
* Desktop & mobile support
* Over 70 global POPs
* Learn more at http://azadify.com/cdn

**Azadify CDN Wordpress Plugin Features**

* Automatically add our CDN to all the pages on your website (i.e. one-click install)
* Manually include specific folders or extensions that you want to use with our CDN
* Manually exclude specific files or extensions that you don't want to use with our CDN
* Enable or disable for HTTPS

**Special Thanks**

CDN Enabler: https://wordpress.org/plugins/cdn-enabler/


== Installation ==
* Download and install the plugin to your Wordpress website
* Go to <a href="http://azadify.com/cdn/signup.php" target="_blank">Azadify CDN's website</a> and register a free Azadify CDN account
* Confirm your email address for Azadify CDN
* Log into <a href="http://azadify.com/cdn/login.php" target="_blank">Azadify CDN dashboard</a> and add your website to Azadify CDN
* Go to your Wordpress dashboard -> Settings -> Azadify CDN and click 'Activate CDN'
* Enjoy the speed!
